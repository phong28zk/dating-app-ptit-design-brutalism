# Dating App - Brutalism + Rounded Design System

## Overview
Build a dating app UI with "Soft Brutalism" aesthetic: neo-brutalist structure (thick borders, hard shadows, bold type) fused with M3-inspired rounded corners and warm romantic color palette. Mobile-first, responsive to 3-column desktop.

## Tech Stack
- Next.js 15 + React 19 + TypeScript
- Tailwind CSS v4 (CSS-first config)
- Framer Motion (animations)
- Figma MCP (design export)
